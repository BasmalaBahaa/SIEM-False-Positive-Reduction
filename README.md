# SIEM False Positive Reduction

## Overview

A Machine Learning-based Security Information and Event Management (SIEM) solution designed to improve threat detection accuracy by reducing false positive alerts.

The system analyzes security logs, extracts events, applies anomaly detection techniques, and assists SOC analysts during incident investigation.

## Key Features

- Direct CSV log analysis
- RAW log analysis
- Security event parsing
- Machine Learning-based alert classification
- False positive reduction
- Alert investigation dashboard
- Log export functionality
- Security reporting

## Problem Statement

Traditional SIEM solutions often generate a large number of false positive alerts, increasing analyst workload and slowing incident response.

This project focuses on improving alert quality using machine learning techniques.
