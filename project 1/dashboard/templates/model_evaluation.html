Metric Formulas
False Positive Rate (FPR)

FPR = FP / (FP + TN)

Out of all normal events, how many were incorrectly flagged as attacks?

False Negative Rate (FNR)

FNR = FN / (FN + TP)

Out of all actual attacks, how many were missed?

Precision

Precision = TP / (TP + FP)

Of all predicted attacks, how many were actual attacks?

Recall (Detection Rate)

Recall = TP / (TP + FN)

Of all actual attacks, how many were detected?

Accuracy

Accuracy = (TP + TN) / (TP + TN + FP + FN)

Overall correctness of the model

False Positive Reduction

Reduction = (FP_old - FP_new) / FP_old × 100
